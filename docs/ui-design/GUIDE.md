2. **MasterGo AI**（备选）
   - 网址：https://mastergo.com/
   - 优势：国产设计工具，支持AI生成
   - 使用方法：同即时设计AI

3. **Google Nano Banana**（如可访问）
   - 网址：https://nanobanana.dev/
   - 优势：专门针对小程序UI设计
   - 使用方法：输入页面布局说明 → 选择小程序模板 → 生成效果图

### 生成步骤

#### 步骤1：准备布局说明

每个页面的详细布局说明已在 `docs/ui-design.md` 第五章"页面布局说明"中提供，包含：
- 页面结构（从上到下的布局）
- 关键元素的位置和尺寸
- 交互状态说明

#### 步骤2：使用AI工具生成

**方法A：使用即时设计AI**

1. 打开即时设计AI：https://js.design/
2. 选择"AI生成"功能
3. 输入Prompt（参考下方模板）
4. 生成效果图
5. 微调细节（如需要）
6. 导出PNG格式，375px宽度

**Prompt模板：**
```
基于以下布局说明，生成微信小程序页面设计：

页面名称：[页面名]
页面结构：
[从ui-design.md中复制页面结构]

关键元素：
[从ui-design.md中复制关键元素说明]

设计规范：
- 主色：#FF6B35（暖橙）
- 字体：14px正文，18px标题
- 圆角：8px卡片，4px按钮
- 间距：16px内边距，24px卡片间距
- 风格：温暖、现代、简洁

请生成符合微信小程序设计规范的页面效果图。
```

**方法B：使用Google Nano Banana**

1. 访问：https://nanobanana.dev/
2. 输入页面布局说明（英文或中文）
3. 选择"小程序"模板
4. 生成效果图
5. 下载PNG格式

#### 步骤3：保存效果图

1. 按命名规范保存：`{模块}_{页面名}.png`
2. 保存到：`docs/ui-design/mockups/` 目录
3. 尺寸：375px宽度（1倍图）

- [ ] 颜色系统符合设计规范（主色#FF6B35）

### 布局检查
- [ ] 页面结构符合布局说明

### 规范检查
- [ ] 符合微信小程序设计规范
- [ ] 导航栏高度正确（44px/48px）
- [ ] 底部Tab符合规范（最多5个）
- [ ] 点击区域足够大（最小44×44px）

## 常见问题

### Q1: AI生成的效果图不符合预期怎么办？

**A:** 
1. 调整Prompt，更详细地描述布局和元素
2. 使用专业设计工具（如Figma）手动调整
3. 参考其他小程序的设计，调整细节

### Q2: 如何确保设计一致性？

**A:**
1. 严格按照设计规范文档执行
2. 使用相同的颜色、字体、间距系统
3. 建立组件库，复用相同组件

### Q3: 效果图需要多高的分辨率？

**A:**
- 1倍图即可（375px宽度）
- 开发时使用rpx单位，自动适配不同屏幕
- 如需切图，可导出2倍图和3倍图

### Q4: 如何与开发团队协作？

**A:**
1. 提供设计规范文档（`ui-design.md`）
2. 提供效果图文件（PNG格式）
3. 提供标注说明（尺寸、颜色、字体）
4. 使用设计协作工具（如Figma）共享设计稿
