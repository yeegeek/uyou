# UI设计效果图

> 微信小程序页面效果图存放目录

## 目录说明

本目录用于存放使用AI工具生成的页面效果图。

## 文件命名规范

格式：`{模块}_{页面名}.png`

示例：
- `discovery_index.png` - 发现首页
- `consultant_detail.png` - 顾问详情
- `order_list.png` - 订单列表
- `partner_workbench.png` - 顾问工作台

## 生成工具推荐

1. **即时设计AI** - 国内可用，支持中文
2. **MasterGo AI** - 国产设计工具
3. **Google Nano Banana** - 专门针对小程序（需翻墙）
4. **Figma AI插件** - 专业设计工具

## 生成步骤

1. 参考 `../ui-design.md` 中的页面布局说明
2. 使用AI工具生成效果图
3. 保存为PNG格式，375px宽度（1倍图）
4. 按命名规范保存到本目录

## 注意事项

- 效果图尺寸：375px宽度（iPhone 6/7/8基准）
- 格式：PNG格式
- 分辨率：1倍图即可（开发时使用rpx适配）
- 命名：使用下划线分隔，小写字母
